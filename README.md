# Renovation Cost Estimator

A React-based tool for estimating renovation costs based on various factors including type of renovation, property type, area size, and scope of work.

## Features

- Calculate renovation costs for different types of projects
- Include wall height in calculations
- Detailed cost breakdown including labor and materials
- IKEA cabinetry baseline for kitchen renovations
- Comprehensive scope of work options
- Responsive design with modern UI

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

## Usage

1. Select renovation type
2. Enter property details
3. Specify area size and wall height
4. Choose finish level
5. Select scope of work items
6. View detailed cost breakdown

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- Node.js
